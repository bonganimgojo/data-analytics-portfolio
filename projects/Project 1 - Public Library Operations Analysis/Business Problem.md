# Project 1: Public Library Operations Analysis
## STEP 1 — Define the Business Problem (Ask)

### Company
City Public Library System

### Problem
The library wants to improve operations and resource allocation by understanding borrowing patterns across branches.

### Stakeholders
* **Library Director:** Wants high-level operational oversight and resource management insight.
* **Branch Managers:** Need granular branch-specific performance metrics.
* **Operations Team:** Focused on day-to-day workflow efficiency and staffing levels.
* **Finance Department:** Interested in cost-efficiencies and budget allocations based on high/low asset utilization.

### Business Questions
1. Which branch serves the most borrowers?
2. Which genres are most popular?
3. Which months have the highest borrowing activity?
4. Which members return books late most often?
5. Which branches have the highest overdue rates?

### Success Criteria
Provide actionable, data-driven recommendations that help the library system improve branch staffing schedules, optimize book inventory planning across locations, and enhance the overall borrower experience.
